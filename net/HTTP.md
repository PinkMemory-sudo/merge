| 404    | 请求内容不存在     |
| 405    | 请求方式错误       |
| 500    | 服务器内部错误     |
| 502    | 网关错误           |
| 503    | 服务不可用         |
| 504    | 网关超时           |
| 505    | 不支持的HTTP版本   |
